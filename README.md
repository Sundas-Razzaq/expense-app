# Expense Tracker

A full-stack Expense Tracker web application built with the MERN stack. It allows users to securely manage their personal finances by tracking income and expenses, viewing analytics, and monitoring their financial activity through an interactive dashboard.

## Live Demo

🔗 https://expense-app-ud4z.vercel.app

## Features

- User Authentication (Register, Login, Logout)
- JWT-based Authentication
- Secure Password Hashing with bcrypt
- Add, Edit, Delete Transactions
- Income & Expense Tracking
- Dashboard Overview
- Analytics & Financial Statistics
- Protected Routes
- Responsive User Interface

## Tech Stack

### Frontend
- React.js
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs
- Joi Validation

### Deployment
- Vercel
- MongoDB Atlas
